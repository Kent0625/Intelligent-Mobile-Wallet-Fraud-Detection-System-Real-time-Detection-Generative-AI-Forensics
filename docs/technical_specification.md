# Final Integrated Project Proposal: Intelligent Mobile Wallet Fraud Detection System

**For:** 3rd-Year Data Science Student
**Focus:** FinTech Fraud Detection (Data Science + Cybersecurity)
**Tech Stack:** Python, Gemini API, Machine Learning
**Timeline:** 6 Weeks

## I. Executive Summary
This project builds a real-time fraud detection system for mobile wallet transactions using:
- **Machine Learning** for pattern-based anomaly detection.
- **Generative AI (Gemini)** for forensic explanation and decision support.
- **Live simulation** to demonstrate production-ready capabilities.

**Key Innovation:** Most fraud systems just flag transactions. Yours explains why using natural language, bridging technical detection with operational decision-making.
